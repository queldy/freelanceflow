# FreelanceFlow

Простое приложение для фрилансеров: управление задачами, клиентами и оплатами в одном месте.

## MVP Функционал
- Регистрация по email/Google
- Добавление задач и проектов
- Учёт оплат (ожидается / оплачено / просрочено)
- Статистика по доходам
- Уведомления о дедлайнах

## Технологии
Flutter + Firebase (Auth, Firestore, Cloud Functions)

## Запуск
1. Склонировать репозиторий
2. Настроить Firebase-проект
3. Запустить `flutter run`
