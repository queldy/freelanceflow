# FreelanceFlow MVP

| Функция                     | Приоритет | Acceptance |
|------------------------------|-----------|------------|
| Регистрация / Вход           | Высокий   | Пользователь может зарегистрироваться и войти, данные сохраняются |
| Создание проектов            | Высокий   | Пользователь может добавить, редактировать и удалить проекты |
| Создание задач               | Высокий   | Можно добавлять, редактировать и удалять задачи, привязывать к проекту и клиенту |
| Учёт клиентов                | Высокий   | Пользователь может добавлять клиентов, редактировать и привязывать к проектам |
| Статус оплаты задач          | Высокий   | Пользователь может отмечать оплату: “ожидается / оплачено / просрочено” |
| Уведомления о дедлайнах      | Средний   | Push-уведомления за 24 ч и 1 ч до дедлайна |
| Статистика доходов и задач   | Средний   | Отображается сумма доходов за месяц и количество выполненных/открытых задач |
| Экспорт CSV                  | Низкий    | Можно выгрузить данные за выбранный период (например, месяц) |
| Offline-режим для задач      | Средний   | Пользователь может работать без интернета, данные синхронизируются при подключении |
| Простая локализация (RU/EN)  | Низкий    | Интерфейс доступен на русском и английском |
